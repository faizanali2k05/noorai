# Demo Script

**Target Time:** 3-5 Minutes

## Intro (0:00 - 0:30)
- "Welcome to NoorAI. In Pakistan, over 350,000 children are on the autism spectrum, yet parents largely search for therapists via Facebook groups."
- "NoorAI changes this. It's an agentic marketplace orchestrated by Google Antigravity."

## Screen 1: Intent & Booking (0:30 - 2:00)
- User enters: *"Mere bete ko speech delay hai 5 saal ka hai Gulberg Lahore mein hafte mein 2 baar 3000 budget"*
- Hit search. Show the ranked therapists.
- Explain the 8-factor Ranking Agent (it's not just distance matching!).
- Proceed to book the therapist and show the Notification Agent output (simulated WhatsApp message).

## Screen 2: Stress Test - The Cancellation (2:00 - 3:00)
- What happens if the therapist cancels at the last minute?
- Trigger cancellation. 
- Show how the Dispute Agent automatically re-discovers a new therapist for the exact same slot and sends an alternative to the user.

## Conclusion (3:00 - 3:30)
- Open the Agent Trace screen showing the exact flow of data through all 7 Antigravity agents.
- Highlight the real-time execution speeds and autonomous error recovery.
